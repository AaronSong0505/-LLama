# skyer: 用最简单的方式训练一个大语言模型

## 安装
从来源安装
```bash
git clone https://github.com/tunglinwood/skyer_llm.git
```
镜像拉取来源路径
```bash
git clone https://hub.nuaa.cf/tunglinwood/skyer_llm.git
```

`pip`安装`skyer_llm`
```python
pip install dist/skyer_llm-0.1.3-py3-none-any.whl 
```
